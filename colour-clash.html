Colour Clash

Overview

Color Clash is a dynamic two-player reaction speed game prototype designed to test and enhance participants' hand-eye coordination through interactive gameplay. The game challenges players to swiftly match colors by pressing corresponding buttons. The prototype integrates user-centered design principles and iterative development processes to optimize usability and gameplay experience.

Design Brief

Create an electronic game that can be played using novel forms of physical interaction.

My Role

My primary responsibility was documentation and user research. I gathered and analyzed user feedback to iteratively improve the game's design and functionality.

Problem Statement

Design and develop an innovative two-player electronic game that enhances hand-eye coordination through novel physical interactions. The game should ensure usability and enjoyment for a diverse audience, incorporating user-centered design principles.

Development Process

  1. Conceptualization:

    Defined core gameplay mechanics inspired by the Batak reaction test, focusing on swift color matching through interactive button presses

  2. Prototyping:

    Created early prototypes using basic materials to test interaction concepts and game mechanics.

  3. Iterative Design:

    Incorporated user feedback to refine physical interaction elements, interface design, and gameplay flow.

  4. User Testing:

    Conducted rigorous testing to gather feedback on gameplay experience, usability, and engagement.

  5. Refinement

    Implemented iterative improvements based on user insights to enhance color differentiation, tactile feedback, and overall accessibility.

  6. Finalization

    Completed the prototype design with improved casing, button layout, and visual/audio feedback systems.

Prototyping

During the development of Color Clash, various prototyping methods were employed to iterate and refine the game concept. This included rapid prototyping with basic materials like cardboard to test initial gameplay mechanics and interaction dynamics.

User Testing and Iteration

Key Insights

I conducted comprehensive user research through playtesting sessions and documented the following key insights:

  • Color Distinction Issues: Users found it challenging to differentiate between cyan and blue on the keypads.
  • Terminology Confusion: Some users were unfamiliar with the term "magenta" and suggested renaming it to "purple."
  • Ergonomic Preferences: Feedback indicated a preference for holding the keypads separately rather than having them fixed on the board.
  • Screen Size Concerns: Several users felt the LCD display was too small.
  • Game Balance: The existing point deduction system created a negative feedback loop for losing players. Users suggested introducing a streak system to balance the gameplay.
  • Sound Feedback: The sounds for correct and incorrect inputs were indistinguishable and perceived as distracting.

Implemented Changes

Based on the user feedback, our team made several iterations to improve the game:

  • Display Pattern Standardization: We standardized the Neotrellis board display to ensure a fair distribution of colors for both players.
  • Keypad Design: Redesigned keypads to be held separately in each hand, enhancing user comfort and versatility.
  • Color Adjustments: Changed cyan to orange for better visibility and renamed "magenta" to "purple."
  • Combo System: Implemented a combo streak system that rewards consecutive correct answers, making the game more balanced and competitive.
  • Sound Feedback: Modified buzzer sounds to clearly differentiate between correct and incorrect inputs.

Final Outcome and Further Improvements

The final prototype incorporated these changes, resulting in a more engaging and user-friendly game. Key features included:

  • Neotrellis Boards: Attached to 3D casings with hidden wires, power supply, and buzzer.
  • LCD Screen: Centrally placed to display scores and colors.
  • Ergonomic Keypads: Players can hold the color boards, enabling two-handed play.
  • Distinctive Colors: Seven easily distinguishable colors (red, green, orange, yellow, purple, blue, white) were used.
  • Enhanced Sound Feedback: Clear audio signals for correct and incorrect inputs.

Future Directions

Looking ahead, several avenues for further improvement and expansion of Color Clash include:

  • Multiplayer Functionality:

    Introducing support for more than two players to enhance social play and accommodate larger groups.

  • Customization Options

    Offering adjustable difficulty levels and gameplay modes to cater to players of varying skill levels and preferences.

  • Enhanced Feedback Mechanisms

    Continuing to refine sound and visual cues to provide clearer and more intuitive feedback during gameplay.
