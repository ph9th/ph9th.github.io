Overview

For my university subject "Internet Programming", I developed a dynamic website for
an online grocery store, providing a comprehensive simulated shopping experience.

GitHub repository

Toolkit

  • HTML, CSS, JavaScript for the front-end
  • PHP for server-side scripting and database interactions
  • AWS Elastic Beanstalk for deployment

Key Features

  • User Interface Design:
    • Designed an intuitive user interface with a custom layout and navigational hierarchy.
    • Created a consistent website logo appearing on all pages.
  • Search and Browsing:
    • Implemented a search box for keyword-based item lookup.
    • Categorized items into sections such as Fruits, Meat, Bakery for easy browsing.
    • Categories are expandable to view all items within them.
  • Item Display:
    • Aligned items in a grid view on pages like the homepage, category pages, and search results.
    • Displayed key information for each item: name, image, unit, price, and stock status.
    • Added buttons beside each item to add them to the shopping cart.
  • Shopping Cart:
    • Designed the shopping cart as a designated area.
    • Made the cart minimized by default, with the option to open it.
    • Allowed users to view, edit, and remove items from the cart.
    • Ensured the cart content persisted for a certain period.
  • Order Placement:
    • Directed users to an order details page for delivery and contact information input.
    • Ensured all delivery details were compulsory for successful order placement.

Deployment and Database

  • Deployed the website using AWS Elastic Beanstalk.
  • Interacted with the database using PHP, including a SQL file for the "products" table.
